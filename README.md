# Ecommerce

Steps to setup the application:
- Clone the repository

For frontend
1. Open Terminal
2. cd .\ecommerce-frontend
2. Run npm install
3. Run npm start

For backend
1. Open Terminal
2. cd .\ecommerce-backend
2. Run npm install
3. Run npm dev

Open the browser:
1. http://localhost:3000/login for frontend
2. http://localhost:5000/ for backend
